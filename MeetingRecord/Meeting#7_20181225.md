## Redis 설치 문제

- CentOS 7에 redis 설치
- VMWare Network bridge mode로 해서 인터넷 설정
- ping의 경우는 network level 3를 체크
- 우리는 http가 되는 걸 봐야 하므로 network level 7이 되는걸 체크해야 함
- 이 명령어가 curl
